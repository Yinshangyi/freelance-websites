# Education / Formation

## ESIEE Paris

- **Degree EN:** Master's degree, Embedded Systems Engineering
- **Degree FR:** Master, Ingénierie des Systèmes Embarqués
- **Date:** 2009 – 2014

### EN

**Specialization:** Embedded Software Engineering with a strong focus on critical systems in the aeronautics, railway, and automotive industries.

The program covered:
- Data Structures & Algorithms, low-level and high-level programming (procedural, OOP, and FP)
- Good software engineering practices: test automation, TDD, SOLID principles, Clean Code, Design Patterns
- Strong emphasis on Java development across multiple layers, from embedded systems to Android and full-stack applications

**Main technologies:**
- Java (core language for several projects, emphasizing object-oriented design)
- Java for Android development and Desktop applications
- C for low-level programming and embedded applications
- Web stack: HTML, CSS, JavaScript for front-end components
- Java for back-end components

### FR

**Spécialisation :** Ingénierie logicielle embarquée avec focus sur les systèmes critiques dans l'aéronautique, le ferroviaire et l'automobile.

Le programme couvrait :
- Structures de données et algorithmes, programmation bas et haut niveau (procédurale, POO et fonctionnelle)
- Bonnes pratiques d'ingénierie logicielle : automatisation des tests, TDD, principes SOLID, Clean Code, Design Patterns
- Fort accent sur le développement Java à travers plusieurs couches, des systèmes embarqués aux applications Android et full-stack

**Technologies principales :**
- Java (langage principal pour plusieurs projets, avec accent sur la conception orientée objet)
- Java pour le développement Android et les applications Desktop
- C pour la programmation bas niveau et les applications embarquées
- Stack web : HTML, CSS, JavaScript pour les composants front-end
- Java pour les composants back-end

## Xidian University

- **Degree EN:** Exchange Program, Computer Science
- **Degree FR:** Programme d'échange, Informatique
- **Date:** 2013 – 2014

### EN

**Specialization:** Computer Science and Software Engineering with a strong focus on programming, algorithms, and system design.

The program covered:
- Computer architecture
- Data compression techniques
- Introduction to Machine Learning: classical algorithms, data processing techniques, and practical applications
- Microservices architecture

**Main technologies:**
- Java (core language for coursework and lab projects focused on system design and algorithmic processing)
- Java for desktop and backend services
- Web stack: HTML, CSS, JavaScript

### FR

**Spécialisation :** Informatique et Ingénierie logicielle avec un fort accent sur la programmation, les algorithmes et le system design.

Le programme couvrait :
- Architecture des ordinateurs
- Techniques de compression de données
- Introduction au Machine Learning : algorithmes classiques, techniques de traitement de données et applications pratiques
- Architecture microservices

**Technologies principales :**
- Java (langage principal pour les cours et projets de lab axés sur le system design et le traitement algorithmique)
- Java pour les services desktop et backend
- Stack web : HTML, CSS, JavaScript
