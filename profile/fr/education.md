# Formation

## ESIEE Paris

- **Diplôme :** Master, Ingénierie des Systèmes Embarqués
- **Date :** 2009 – 2014

**Spécialisation :** Ingénierie logicielle embarquée avec un fort accent sur les systèmes critiques dans l'aéronautique, le ferroviaire et l'automobile.

Le programme couvrait :
- Structures de données et algorithmes, programmation bas et haut niveau (procédurale, POO et fonctionnelle)
- Bonnes pratiques d'ingénierie logicielle : automatisation des tests, TDD, principes SOLID, Clean Code, Design Patterns
- Fort accent sur le développement Java à travers plusieurs couches, des systèmes embarqués aux applications Android et full-stack

**Technologies principales :**
- Java (langage principal pour plusieurs projets, avec accent sur la conception orientée objet)
- Java pour le développement Android et les applications Desktop
- C pour la programmation bas niveau et les applications embarquées
- Stack web : HTML, CSS, JavaScript pour les composants front-end
- Java pour les composants back-end
